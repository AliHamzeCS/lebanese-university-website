# Lebanese University Website 🎓

A modern university website interface designed using HTML5 and CSS3 featuring a landing page and admission application form.

![Home Page Screenshot](images/home.png)

## 📌 Overview

This project is a university landing page with a modern dark theme design.  
It provides an attractive homepage and an online application form interface for students.

## 📄 Pages

### Home Page

![Application Form](images/apply.png)

- Modern navigation bar
- Hero section introducing the university
- University features section
- Research and academic information
- Footer with contact information

### Application Page
- Student personal information form
- Academic history section
- Program selection
- Styled application button

## ✨ Features

- Modern dark UI design
- Gradient backgrounds
- CSS animations
- Hover effects
- Sticky navigation header
- Responsive form layout
- Clean user interface

## 🛠 Technologies Used

- HTML5
- CSS3
